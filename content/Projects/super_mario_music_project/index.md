## 📝 General Description
### Project Overview
This 3D platformer prototype was developed in collaboration with **Level Up Lab**, a student team at **Politecnico di Torino** specializing in game development. 

The project was a collective effort where I served as the **Lead Programmer**. I worked alongside three talented teammates who managed the music production, artistic direction, and project coordination. Our goal was to blend classic Nintendo aesthetics with innovative rhythmic mechanics.

### Key Features
* **Reimagined Mechanics:** We drew inspiration from titles like *Super Mario Odyssey* and *Super Mario 3D World*, implementing a fluid 3D movement system.
* **Rhythm-Based Gameplay:** The core innovation lies in the integration of rhythm game elements. Players must time their jumps and movements to match the beat on dynamic, rhythm-synced moving platforms.
* **Musical Mushroom Power-up:** A unique "Musical Note Mushroom" bonus that allows players to temporarily trigger a slow-motion effect (time dilation), making complex rhythmic sections more manageable.

---

## 🛠️ Technical Details
As the programmer, I was responsible for:
* Developing the 3D character controller and physics.
* Implementing the synchronization system between the game environment and the audio tracks.
* Coding the power-up logic and the time-manipulation mechanics using GDScript.
